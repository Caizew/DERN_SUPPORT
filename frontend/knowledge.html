Diagnosing Slow Computer Performance

Symptoms

  • Applications take too long to open
  • System freezes or lags under load
  • Disk or CPU usage spiking to 100%

Step-by-Step Fix

  1. Check Task Manager / Activity Monitor: Identify any processes consuming excessive CPU or RAM.
  2. Clear Disk Space: Remove temporary files, uninstall unused programs, and empty Recycle Bin.
  3. Disable Startup Programs: Use Task Manager (Windows) or System Preferences → Users & Groups → Login Items (Mac) to turn off unneeded startup apps.
  4. Run Disk Defragmentation (Windows only): Open “Defragment and Optimize Drives” and optimize your main drive.
  5. Check for Malware: Run a full system scan with an up-to-date antivirus tool.
  6. Upgrade Hardware (Optional): If performance remains poor, consider upgrading to SSD or adding more RAM.

Fixing Common Wi-Fi Connectivity Issues

Symptoms

  • Cannot connect to Wi-Fi network
  • Connection drops intermittently
  • Slow wireless speeds despite strong signal

Step-by-Step Fix

  1. Restart Router & Modem: Power cycle both devices (unplug 30 seconds, then plug back in).
  2. Forget & Reconnect to Network: In settings, “Forget” the wireless network and then re-enter password.
  3. Update Wireless Drivers: On Windows, open Device Manager → Network adapters → Update driver. On Mac, ensure macOS is up to date.
  4. Check Channel Interference: Use a Wi-Fi scanning tool (like inSSIDer) to pick the clearest channel on 2.4GHz or 5GHz.
  5. Position Router Optimally: Place it centrally, off the floor, away from microwaves and thick walls.
  6. Reset Network Settings (If Needed): Windows: `Settings → Network & Internet → Status → Network reset` Mac: Delete and re-add Wi-Fi service in Network Preferences.

Removing Malware and Viruses Safely

Symptoms

  • Unexpected pop-ups or ads
  • Performance slows after opening certain websites
  • Unknown programs running in background

Step-by-Step Fix

  1. Run Safe Mode Scan (Windows): Restart in Safe Mode (F8 on startup), then run antivirus scan.
  2. Use a Bootable Antivirus USB: Create a bootable rescue disk (e.g., Kaspersky Rescue Disk) and scan before Windows boots.
  3. Install & Update a Reputable Antivirus Tool: Examples: Malwarebytes, Avast, Bitdefender. Perform a full system scan.
  4. Remove Quarantined Files: Follow antivirus prompts to delete or quarantine infected files.
  5. Check Browser Extensions/Add-ons: Remove suspicious extensions and reset browser settings to default.
  6. Enable Windows Defender/Mac XProtect: Ensure built-in protections are enabled and updated regularly.
